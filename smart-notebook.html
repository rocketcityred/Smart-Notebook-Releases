              Smart Notebook — Guide
              
      
    
              
    
              

      
              
        Smart Notebook is a single offline HTML file that captures emails, lets you write
        freeform notes, tag and prioritize both, set reminders, search everything in real
        time, and print or back up your data. It runs entirely from your browser; nothing
        leaves your machine.
      
              

      
              Read this first — what works and what doesn't
              
      
                
        
              • No closed-tab reminders. Reminders fire only while this tab is open. Close the tab and the schedule pauses.
                • 
        
              • No desktop / OS notifications on most browsers when opened from a local file. The in-app banner + tab-title flash + beep is the alert mechanism.
                • 
        
              • Browser storage isn't guaranteed for local files — export to JSON regularly. That is your only real backup.
                • 
        
              • Drag-and-drop from Outlook isn't possible. Use copy/paste.
                • 
        
              • Email attachments aren't captured. Only the text body comes across via the clipboard.
                • 
        
              • No network calls. Disconnect your network and the app continues to work identically. That's the design.
                • 
      
              

      
              The main view
              
      
                
        
              • The left pane is a unified list of every item — captured emails AND freeform notes — interleaved newest-first. Each row carries an EMAIL or NOTE badge so the kind is obvious.
                • 
        
              • The right pane shows the detail of the selected item with editable fields.
                • 
        
              • Click any row to open it in the detail pane.
                • 
        
              • Close the detail with the ✕ Close button at the top of the toolbar, by pressing Esc (when no text field is focused), or by clicking the same row in the list again.
                • 
        
              • A status line at the bottom shows ready state and the reminder that your data is local.
                • 
      
              

      
              Adding content
              
      
              Capture an email (top-bar blue + Capture button):
              
      
                
        
              1. Open the email in Outlook → Select All (Ctrl+A) → Copy (Ctrl+C).
                2. 
        
              2. In Smart Notebook, click + Capture, click the dashed paste area, and paste (Ctrl+V).
                3. 
        
              3. The app guesses Subject / From / To / Cc / Sent date — correct anything wrong before saving.
                4. 
        
              4. Optionally pick categories and a priority (both default to none).
                5. 
        
              5. Click Save email.
                6. 
      
              
      
              The full pasted text is preserved exactly as-is and is forever read-only — it's the captured email's source of truth. The editable Body field below it is a separate copy you can revise without losing the original.
              

      
              Add a freeform note (top-bar Notes button):
              
      
                
        
              • Type a title (optional) and text. Pick a priority (default None). Attach labels.
                • 
        
              • Click Save note. It appears in the main list with a green NOTE badge.
                • 
        
              • You can also edit any existing note straight from the main list — click the row, edit in the right detail pane.
                • 
      
              

      
              Organizing — categories, labels, priority, done
              
      
                
        
              • Categories tag captured emails. Manage from the top-bar Categories button.
                • 
        
              • Labels tag freeform notes. Manage from the top-bar Labels button, or via Manage labels inside the Notes modal.
                • 
        
              • Categories and labels are deliberately separate vocabularies so quick-jot labels don't pollute structured email categories.
                • 
        
              • Deleting a category or label keeps the items that used it — it just removes the tag from them.
                • 
      
              

      
              Assigning tags to an item (chip picker in the detail pane):
              
      
                
        
              • Currently-assigned tags appear as colored chips, each with an ×.
                • 
        
              • Click × on a chip to remove that tag — saved instantly.
                • 
        
              • Pick from the — Add category — / — Add label — dropdown to add an unassigned one — saved instantly.
                • 
        
              • Clear all removes every assigned tag at once — also saved instantly.
                • 
        
              • Tag changes do NOT require the Save changes button. That button covers the text fields (subject, body, title, priority, etc.).
                • 
        
              • In the Notes editor and the freeform-note detail, the Create new label row below the picker creates a brand-new label and attaches it in one step.
                • 
      
              

      
              Priority and Done apply to every item type (emails, notes, reminders):
              
      
                
        
              • Priority: High / Medium / Low / None. New items default to None. None-priority items show no priority pill in the list, to keep it quiet.
                • 
        
              • Mark done / Mark not done toggles in the detail toolbar. Done items get a green "done" badge and are dimmed with a strikethrough but stay fully searchable.
                • 
        
              • Marking an email done never alters its captured raw text — done is metadata.
                • 
      
              

      
              Search and filters
              
      
                
        
              • The search box in the top bar filters emails AND notes in real time as you type (~80ms debounce). Matches are highlighted in titles and previews.
                • 
        
              • Enter in the search box opens the top filtered match.
                • 
        
              • The × at the right of the search box clears it instantly and refocuses for a new query.
                • 
        
              • Combine with the Priority, Category, and Done dropdowns and the Sort mode:
          
                  
            
                • Priority applies to both emails and notes (both carry a priority).
                  • 
            
                • Category applies only to emails — selecting one filters notes out.
                  • 
            
                • Done defaults to Active (hide done). Switch to All or Done only.
                  • 
            
                • Sort options: Newest / Oldest / Priority (High→Low) / Subject A→Z.
                  • 
          
                
        
                • 
      
              
      
              Notes-modal rich filters (open Notes, scroll past the editor): a richer filter strip for notes specifically — multi-label AND, date range over the note's created date, plus priority/done/sort. All combinable. Clear all filters resets the strip.
              

      
              Reminders
              
      
                
        
              • Open Reminders from the top bar to add one — standalone, linked to an email, or linked to a note (pick from the linkage dropdown).
                • 
        
              • From any email or note's detail pane, click Add reminder to prefill the linkage and a sensible title.
                • 
        
              • Each reminder has a title, due date/time, optional notes, and a priority.
                • 
        
              • When due, the app raises three things at once: a persistent banner at the top of the screen, a flashing tab title ((2) DUE — Smart Notebook), and a short beep (toggle the sound in the Reminders modal — default on).
                • 
        
              • From the banner or the Reminders modal: Done acknowledges (the unified "done" state), Dismiss closes the banner without completing, Snooze 10m pushes the due time out.
                • 
        
              • Items with a future reminder show a ⏰ clock chip in the list with the next due time. Overdue chips are colored red.
                • 
        
              • Deleting an email or note that has attached reminders prompts you to either delete the reminders too or keep them (detach).
                • 
      
              

      
              Printing
              
      
                
        
              • Open any email or note in the detail pane, then click Print in the toolbar.
                • 
        
              • The browser's print dialog opens with a clean, formatted view of just that item — your app chrome is hidden.
                • 
        
              • Pick a printer, or choose Save as PDF from the destination dropdown to save it as a PDF you can email.
                • 
        
              • The printed page shows the subject/title, metadata, body, labels/categories, attached reminders, and a footer with the print timestamp.
                • 
      
              

      
              Data management (the DATA group)
              
      
              The four buttons inside the bordered DATA group in the top bar are administrative — used occasionally, not constantly.
              
      
                
        
              • Backup downloads a JSON file containing every record. The filename is dated (for example smart-notebook-backup-2026-05-25-1430.json). Do this often — it's the only real backup.
                • 
        
              • Restore reads a backup file. The file is verified (shape + record counts + checksum) BEFORE any change is made. Pick a mode:
          
                  
            
                • Merge — add records whose id is new; existing ones kept.
                  • 
            
                • Merge with overwrite — add new and overwrite matching ids.
                  • 
            
                • Replace all — wipe current data and load the backup. A safety backup of your current data is automatically downloaded first.
                  • 
          
                
        
                • 
        
              • Verify validates a backup file without restoring it — use this to confirm a backup is intact before relying on it.
                • 
        
              • Older backup formats are automatically migrated forward on import (for example, an export from before priority existed gets None backfilled).
                • 
        
              • If it's been more than 7 days since your last backup, a non-blocking nudge appears at the top of the app suggesting you export.
                • 
      
              

      
              Cleanup — keep the app snappy
              
      
              Open Cleanup (inside the DATA group) to bulk-purge items. Each section previews the count before you confirm. A safety backup is downloaded first by default (toggle at the bottom of the modal).
              
      
                
        
              • Purge done items — delete items currently marked done (per-type checkboxes, optional minimum age).
                • 
        
              • Purge old items — delete items created more than N days ago, optionally only those already done.
                • 
        
              • Purge by category — delete every email in a chosen category; optionally also delete the category itself afterward.
                • 
        
              • Purge by label — delete every note with a chosen label; optionally also delete the label itself.
                • 
        
              • Reminder cleanup — delete old acknowledged/dismissed reminders.
                • 
      
              
      
              Deleting an email or note auto-removes any reminders linked to it, so the database never has orphaned reminders. The app stays responsive up to a few thousand items; past that, periodic cleanup helps.
              

      
              Keyboard shortcuts

              Enter in searchOpen the top filtered result
              
        
              EscClose the topmost modal — or, if no modal is open and no text field is focused, close the open detail item
              
        
              Click an itemOpen it in the detail pane
              
        
              Click the same item againClose it
              
        
              Enter in New label nameCreate the new label and attach it
              
      
              

      
              Defaults you can rely on
              
      
                
        
              • New emails and notes start with no categories, no labels, and priority None. Nothing is preassigned.
                • 
        
              • You can add or remove categories, labels, and priority at any time. Tag changes are saved instantly; priority changes need Save changes.
                • 
        
              • Done filter defaults to Active (hide done) — done items stay searchable but stop cluttering the list.
                • 
      
              

      
              Where is my data?
              
      
              Your data lives in this browser's IndexedDB, scoped to the path you opened the HTML file from. There is no file on disk you can copy. To move data between machines, Backup on one and Restore on the other. If you copy the HTML file to another machine, the file works there but the data does NOT travel with it.
              

      
              Security and privacy
              
      
                
        
              • The app makes zero network calls — verify with the browser DevTools Network tab if you like.
                • 
        
              • Pasted email HTML is sanitized before display: scripts, event handlers, and external resource references (including <img src>) are stripped so a malicious email can't beacon out.
                • 
        
              • Stored data is NOT encrypted at rest. Anyone with access to your browser profile can read it.
                • 
        
              • Backup JSON files contain all your content unencrypted. Store them as carefully as you'd store the original emails.
                • 
      
              

      
              Troubleshooting
              
      
                
        
              • My data disappeared. Likely the browser cleared local storage, or you opened the file from a different path / profile. Restore from your last JSON backup. Export often.
                • 
        
              • A reminder didn't fire. The tab must be open at the due time. If it was, check the sound toggle in the Reminders modal and confirm the system clock hasn't been changed.
                • 
        
              • Pasting an email left fields blank. The heuristic parser is intentionally conservative — it leaves fields blank rather than guess wrong. Fill them in manually before saving; the full raw text is still preserved and searchable.
                • 
        
              • The "formatted" view of an email looks plain. HTML is sanitized for safety; scripts, event handlers, and external image URLs are stripped on purpose.
                • 
        
              • Restore says "checksum mismatch". The backup file is corrupted or was edited. The import is refused so your current data stays intact. Use a different backup.
                • 
        
              • Restore says "schemaVersion newer than this app supports". The backup was made by a newer Smart Notebook build. Use that newer build to restore it.
                • 
        
              • The print preview is blank. Reload the page (Ctrl+Shift+R) and try again — the print machinery hides app chrome via @media print, so the document needs to be freshly loaded.
